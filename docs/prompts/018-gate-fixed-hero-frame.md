# Gate Handoff: Fixed Hero Frame

Gate review scope:

- Review the owner handoff in `docs/prompts/016-owner-fixed-hero-frame.md`.
- Review the tester handoff in `docs/prompts/017-tester-fixed-hero-frame-review.md`.
- Review tester evidence in `docs/reviews/011-fixed-hero-frame-review.md`.
- Decide whether a correction handoff is required.

Gate criteria:

- The hero carousel frame has stable sizing and does not change with image dimensions.
- Images are shown fully inside the frame.
- Unused image area is filled by black background/letterboxing.
- An outside border line is present.
- Existing carousel controls and gallery link behavior remain intact.
- No release permission is granted unless explicitly approved separately.
